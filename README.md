# Guía de Markdown — Comandos y Ejemplos

Este documento explica cómo usar Markdown con ejemplos básicos y otros un poco menos comunes pero fáciles de entender.  
Markdown sirve para dar formato al texto usando símbolos simples.

---

## 1. Encabezados (Títulos)

```markdown
# Título 1
## Título 2
### Título 3
#### Título 4
```
## 2. Negritas, cursivas y tachado
```markdown
**Negritas**
*Cursiva*
~~Tachado~~
***Negrita y cursiva a la vez***
```

## 3. Listas
```markdown
- Manzana
- Pera
- Mango
```
## Lista numerada
```markdown
1. Paso uno
2. Paso dos
3. Paso tres
```
## Lista anidada
```markdown
- Lenguajes
  - Python
  - JavaScript
    - Node.js
```
## 4. Enlaces
```markdown
[Visitar GitHub](https://github.com)
```
## 5. Citas y citas anidadas
```markdown
> Esta es una cita
>> Esta es una cita dentro de otra
```
## 6. Tablas
```markdown
| Producto | Precio | En Stock |
|---------|--------|----------|
| Manzana | $10    | Sí       |
| Pera    | $12    | No       |
```
